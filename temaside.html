TEMA 2

Grundlæggende Web


På uddannelsens første tema, blev vi introduceret for faglige elementer til brug, inden for grundlæggende web. Vi blev på dette tema, introduceret for produktion og design af digitale og responsive brugergrænseflader.

Vi blev på dette tema introduceret for de digitale sprog inden for DOM’en (Document Object Model), HTML (Hypetext Mark up Language), og CSS (Cascading Style Sheets), hvilket er sprog der kan benyttes til at opbygge en visuel hjemmeside. HTML er tekstbaseret visning, mens sproget inden for CSS, styler den måde som elementerne i HTML’en er skrevet, ved at give det en form for udseende.

Ydermere blev vi også introduceret for brugen af Photoshop og Adobe XD, da vi i dette tema skulle udvikle vores første website med tilhørende splashbillede og stockphotos, som skulle tilpasses i størrelse og udvælges efter en given stilart.

I dette tema, lærte vi desuden også at uploade via ftp klienten Zilezilla, og erhvervede vores eget webhotel, til eget domæne.
